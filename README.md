# Environment variables needed
```bash
PUBLIC_SUPABASE_URL=
PUBLIC_SUPABASE_ANON_KEY=
```

# Start the app
```bash
pnpm install
pnpm dev
```
# Build the app
```bash
pnpm build
# To preview the production build
pnpm preview
```
